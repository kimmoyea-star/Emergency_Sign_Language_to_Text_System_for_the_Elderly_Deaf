# Emergency_Sign_Language_to_Text_System_for_the_Elderly_Deaf

기존 응급 신고 체계의 한계 : 현재 112 및 119 신고 시스템은 음성 통화 위주이며, 문자/앱 신고 방식은 디지털 조작과 타자 입력에 익숙하지 않은 고령 농인에게 긴박한 상황에서 생명과 직결되는 높은 진입 장벽임. 또한 기존 영상 통화를 통한 수어 신고는 119 상황실 요원이 수어를 모를 경우 통역사를 거쳐야 하므로 골든타임이 지연되며, 통신 환경에 따라 영상이 끊기는 문제가 발생함. 고령 농인이 스마트폰 카메라에 핵심 응급 수어(가슴 통증, 화재 등)를 취하면, 기기 내부의 AI가 이를 즉시 텍스트로 번역하여 119 상황실 모니터에 전송하는 시스템을 제안함. 이를 통해 가벼운 텍스트 전송만으로 가장 빠르고 정확한 무장애(Barrier-free) 구조 요청 환경을 구축하고자 함.

VS Code와 아나콘다(python=3.10) 사용

데이터 실시간 촬영

인공지능 모델 학습 (Training): 병합된 data_all.csv를 로드하여 LSTM 신경망을 가르치고 가중치 두뇌 파일을 생성함. (40프레임 최적화 반영) 실행 완료 시 최적의 가중치가 압축된 두뇌 파일인 sign_model.pth가 자동 생성/업데이트됨.

실시간 수어 추론 및 텔레그램 관제 (Testing): 웹캠을 활성화하여 사용자의 수어를 실시간 트래킹하고, 확신도 80% 이상 + 1초 이상 유지 조건 만족 시 외부 API를 통해 위치 정보와 함께 텔레그램 긴급 알림을 발송함. 실행 전에 test_model.py 파일 상단의 TELEGRAM_TOKEN 및 TELEGRAM_CHAT_ID에 본인의 봇 정보를 입력해야 정상적으로 폰에 알림이 수신됨. 웹캠 창을 선택한 상태에서 키보드 ESC 키를 누르면 안전하게 종료됨.
